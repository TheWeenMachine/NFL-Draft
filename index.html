About This Report

How This Report Works

This report combines a 300-player consensus draft board with Pro Football Focus (PFF) statistical data from the 2025 college season, a separate PFF run-blocking export, DraftBuzz scouting reports, and PFF Big Board season grades. Each prospect is scored using a position-specific composite — 5 metrics for QB, WR, and TE; 4 metrics for all other positions.

All composite scores are percentile ranks — a score of 80 means the player graded better than 80% of all players at their position in the PFF database (12,900+ players). The DraftBuzz rank, PFF Big Board rank, and consensus rank are three independent evaluation systems — divergence between them is where the most interesting draft analysis happens.

PFF Composite = Σ ( Metric Percentile × Weight ) / Σ Weight
Inverted metrics (lower=better): Percentile = 100 − searchsorted(pool, val, side="left") / n × 100
Run block grades sourced from separate PFF offense run-blocking CSV export

Limitations

What This Report Doesn't Measure

Athletic Testing: Combine results not directly scored in PFF composite (physical measurements shown via DraftBuzz).

Film Qualities: Pad level, hand technique, route refinement, and pre-snap processing are outcome-agnostic and not captured by PFF data.

Scheme Fit: Percentile pool includes all college systems; spread vs. pro-style variance is not adjusted for.

Character & Medical: Not reflected in any score. Teams regularly move prospects based on these factors.
